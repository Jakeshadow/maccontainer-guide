What This Is Not

Not for x86/Intel Mac users

Apple Container requires Apple Silicon. Intel Macs are not supported.

Not for pre-macOS 26 users

Requires macOS 26's Virtualization.framework. Older macOS versions can't run Apple Container.

Not a Docker-bashing manifesto

Docker is excellent — especially on Linux. This guide is for macOS developers who choose to switch.

Not a replacement for reading the source

We link to apple/container GitHub throughout. This manual is a practical companion, not a substitute for understanding the tool.
